# TaskBunny App

A modern Flutter-based errand marketplace that connects customers with nearby workers for everyday tasks. The platform includes dedicated applications for Customers, Workers, and Administrators, powered by Firebase services.

---

# Features

## Customer App
- User registration and login
- Create errand requests
- Upload task images
- Live location selection
- Browse nearby workers
- Real-time task status updates
- In-app notifications
- View task history
- Rate completed workers

---

## Worker App
- Worker registration
- View available nearby tasks
- Submit bids for errands
- Accept assigned jobs
- Navigation to customer location
- Complete assigned tasks
- Earnings dashboard
- Subscription payment system
- Task history
- Real-time notifications

---

## Admin App
- User management
- Worker approval
- Subscription payment approval/rejection
- Block or unblock workers
- Monitor platform activity
- Dashboard analytics
- Manage reports

# Subscription System

Workers contribute a configurable percentage of their completed task earnings as a subscription fee.

Features include:

- Automatic earnings calculation
- Payment submission
- Admin approval workflow
- Payment rejection handling
- 30-day subscription period after approval
- Automatic reminders before expiration
- Worker suspension support for overdue subscriptions
